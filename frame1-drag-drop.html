Interaction Notes

State A · Default
  • Dashed SVG border marches clockwise via stroke-dashoffset animation at ~1.8s loop
  • Subtle pulse ring radiates outward every 2.6s to draw the eye
  • Document icon floats gently on a 3s ease-in-out cycle
  • Green glow on the icon hints at AI-powered action
  • Browse Files button acts as fallback CTA; triggers OS file picker
  • Format chips confirm accepted file types at a glance
State B · Active Hover
  • Triggered on dragenter — entire drop zone border switches to solid green
  • Box-shadow blooms to a multi-layer green glow (no JS needed on demo)
  • Background of the app area gets a subtle radial green tint
  • Document icon scales up and bounces on ~1.1s loop to show "ready"
  • 12 CSS-only particles (8 dot sparks + 4 star glyphs) float upward at staggered delays
  • Text switches to "Release to start analysis" — clear affordance
State C · Processing
  • Upload card replaces drop zone; shows PDF cover-page thumbnail mockup
  • Progress bar animates from 0→42% on page load via CSS forwards fill
  • Shimmer effect loops on the filled bar after initial fill completes
  • Extraction steps fade-slide in sequentially with animation-delay stagger
  • Step 1 shows ✓ done; step 2 shows spinning ⟳ + animated ellipsis dots
  • Step 3 shows ⏳ queued — communicates AI pipeline depth
