Design Annotations
→ Address auto-extracted from OM via LLM — user confirms, doesn't re-type
→ Overture Maps provides open building footprint data — no paid tile API needed
→ State A fly animation masks geocoding latency (~1–2s)
→ State B footprint glows to indicate clickability; tooltip on hover
→ State C solid fill + toast gives instant visual confirmation
→ 94% confidence score from Overture — low confidence surfaces manual override
→ Building details pre-populated (area, floors, year) → skip manual entry in later frames
→ "Skip — enter coordinates manually" escape hatch for edge cases
